Rootfall - SAE601
Projet réalisé dans le cadre de la SAE601.
Jeu d’exploration 2D en monde semi-ouvert développé avec Godot 4.

-----------------------------------------------------------

Description
Rootfall est un jeu 2D dans lequel le joueur explore un environnement interactif afin de collecter des éléments et progresser dans le monde. Le gameplay repose sur l’exploration, l’interaction avec l’environnement et la découverte progressive de zones.
Le projet intègre notamment :
•	déplacement du joueur
•	caméra dynamique
•	cycle jour / nuit influençant le gameplay•	interactions et collecte d’objets
•	interface utilisateur
•	système de sauvegarde (version locale)

-----------------------------------------------------------

-----------------------------------------------------------

Système de sauvegarde
Le projet inclut un système de sauvegarde permettant :
•	création de plusieurs parties
•	chargement et reprise de progression
•	suppression de parties
•	renommage des sauvegardes
•	gestion de slots multiples
Ce système fonctionne intégralement en version locale (desktop).
